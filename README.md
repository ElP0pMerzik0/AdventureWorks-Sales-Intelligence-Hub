# AdventureWorks Sales Intelligence Hub 📊

## Project Overview
A comprehensive Sales Dashboard built using **Excel Power Pivot** and **DAX**, transforming the classic AdventureWorks dataset into actionable business insights.

## 🛠️ Tech Stack & Skills
- **Data Source:** SQL Database (AdventureWorks).
- **ETL:** Power Query (Cleaned and Transformed 7+ tables).
- **Data Modeling:** Star Schema design with a custom `Dim_Date` table.
- **Analysis:** DAX Measures & Pivot Tables.
- **Visualization:** Interactive Dashboard with UI/UX focus (Dark Theme).

## 🧩 Data Model
The project follows a **Star Schema** architecture to ensure performance and scalability.
![Data Model](Documentation/image_ae3c2a.png)

## 🚀 Features
- **Dynamic Filtering:** Slicers for Category, Subcategory, Color, and Region.
- **Time Intelligence:** Integrated timeline for monthly performance tracking.
- **Key Metrics:** Real-time calculation of Revenue, Tax, Shipping Costs, and Order Volume.
